Plantilla SEO para nuevos artículos

Formato base para mantener consistencia editorial, interlinking y utilidad práctica en toda la sección.

Estructura mínima obligatoria

  1. Title + meta description orientados a intención de búsqueda real.
  2. H1 único + introducción accionable.
  3. Bloque “qué aprenderás”.
  4. Contenido central con ejemplos y checklists.
  5. Bloque legal/normativo (si aplica) con fuente oficial.
  6. CTA a búsqueda/alerta + enlaces a guía relacionada.
  7. FAQ schema con 2–4 preguntas reales.

Interlinking recomendado

  • Enlace al hub: Ruta de ingreso.
  • Enlace a glosario: Glosario laboral público.
  • Enlace a guías por etapa (búsqueda, CV, postulación, seguimiento).
  • Enlace de conversión: crear alerta.
