import asyncio, json

from scrapers.base import HttpClient

    inst=next(i for i in json.load(f)['instituciones'] if i['id']==146)

    s=TrabajandoCLScraper(fuente_id=146,institucion=inst)

## Resultado

### 1) Gatekeeper / routing
- `run_all` reporta bypass explícito para la fuente 146:
  - `kind=custom_trabajando`
  - `scraper_external_ats / ats_trabajando`
- Con esto se confirma que el routing de la fuente quedó correcto.

### 2) Extracción real
- En este entorno no hay conectividad efectiva hacia `fiscaliadechile.trabajando.cl`.
- El scraper reintenta 3 veces y retorna `offers_count=0` con error de red:
  - `Cannot connect to host fiscaliadechile.trabajando.cl:443 ... [Network is unreachable]`

## Conclusión técnica
- **Routing:** OK (confirmado).
- **Extracción E2E en este entorno:** No concluyente por limitación de red saliente.
- **Acción siguiente recomendada:** correr el mismo smoke en runner con salida a internet (staging/prod) y revisar `offers_count`, `scraper_runs` y persistencia en `source_evaluations`.
